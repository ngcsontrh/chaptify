Chaptify

Tool tách nội dung file .epub ra thành các file .txt riêng lẻ theo chương.

Cách dùng:

- Chạy chương trình (Chaptify.exe)
- Chọn chức năng Extract
- Nhập đường dẫn file epub
- Tool sẽ tự tạo thư mục chứa các file text ở cùng chỗ với file gốc

Lưu ý:

- Chỉ hỗ trợ file .epub chuẩn
- File text xuất ra là UTF-8
